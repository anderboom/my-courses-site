Информация о курсе

Для кого данный курс?
Для кого предназначен данный курс?

В первую очередь для учеников старших классов, которые собираются поступать в технические ВУЗы по специальностям, связанным с программированием, а также для студентов данных специальностей. Если вы решили попробовать себя в новой для себя профессии веб-разработчика и получить базовые навыки — добро пожаловать на данный курс! Основные требования для учеников — базовый английский и желание учиться и осваивать шаг за шагом новую специальность.

О курсе. Данный курс будет состоять из двух больших модулей. Первый модуль называется «Базовый» , второй модуль - «Разработка full stack приложений» или, как я его назвал в шутку - «Курс для выживших)))». Продолжительность каждого курса — 6 месяцев. Как будут построены занятия. Занятия проводятся группами по 4 - 5 человек два раза в неделю по 2 урока каждое(45 мин. урок). Важно понимать, что вы не просто группа №1, 2, 3 … - вы одна команда. Для каждой группы будет создана отдельная группа в Telegram для общего обсуждения пройденного материала и домашних заданий. Если кто-то что-то не понял, в первую очередь попытайтесь разобраться в данном вопросе при помощи учеников из вашей команды. После каждого занятия вы будете получать домашние задания для понимания и закрепления пройденного материала. Некоторые домашние задания будут на английском, для дополнительной мотивации изучения английского языка. Весь процесс обучения с первого занятия будет проводиться с использованием реальных программ, которые используют реальные разработчики. Не будет никаких «песочниц», встроенных площадок — всё как в реальном процессе разработки. Немного забегу вперёд и ещё раз повторюсь что вы — команда. При изучении второго модуля, когда будем писать небольшое full stack приложение, мы разделимся на 2 группы по 2 человека. Одна группа будет писать backend (серверную часть приложения) и предоставлять документацию по запросам на сервер, вторая группа будет писать frontend (клиентскую часть). Потом группы поменяются местами. Надеюсь, что такой формат обучения будет интересным и познавательным.

В первом модуле мы будем изучать HTML, CSS, Java Script и в конце курса мы подключим к нашему проекту библиотеку Bootstrap, адаптируем для просмотра на различных устройствах, чтобы наш проект начал выглядеть ну очень красиво). В процессе обучения мы научимся работать с системой управления версиями Git и опубликуем свой проект на GitHub pages. Что это будет за проект? В процессе изучения первого модуля мы напишем и опубликуем резюме и свой личный справочник  по языку разметки гипертекста HTML, языку описания внешнего вида CSS и основам одного из самых популярных и востребованных языков веб разработки — Java Script. По окончании курса обучения будет проведено собеседование с задачами и вопросами по пройденному материалу для проверки и закрепления полученных знаний.
