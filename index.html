Інформація про курс

Для кого цей курс?
Для кого призначено цей курс?

Насамперед для учнів старших класів, які збираються вступати до технічних ВНЗ за спеціальностями, пов'язаними з програмуванням, а також для студентів цих спеціальностей. Якщо ви вирішили спробувати себе в новій для себе професії веб-розробника і отримати базові навички - ласкаво просимо на цей курс! Основні вимоги для учнів — базова англійська та бажання вчитися та освоювати крок за кроком нову спеціальність.

Про курс. Цей курс складатиметься з двох великих модулів. Перший модуль називається «Базовий» , другий модуль - «Розробка full stack додатків» або, як я його назвав жартома - «Курс для тих, хто вижив)))». Тривалість кожного курсу – 6 місяців. Як буде збудовано заняття. Заняття проводяться групами по 4 - 5 осіб двічі на тиждень по 2 уроки (45 хв. урок). Важливо розуміти, що ви не просто група №1, 2, 3… – ви одна команда. Для кожної групи буде створено окрему групу в Telegram для загального обговорення пройденого матеріалу та домашніх завдань. Якщо хтось щось не зрозумів, насамперед спробуйте розібратися в цьому питанні за допомогою учнів із вашої команди. Після кожного заняття ви отримуватимете домашні завдання для розуміння та закріплення пройденого матеріалу. Деякі домашні завдання будуть англійською для додаткової мотивації вивчення англійської мови. Весь процес навчання з першого заняття буде проводитись із використанням реальних програм, які використовують реальні розробники. Не буде жодних «пісочниць», вбудованих майданчиків – все як у реальному процесі розробки. Трохи забіжу вперед і ще раз повторюся, що ви — команда. При вивченні другого модуля, коли писатимемо невеликий full stack додаток, ми розділимося на 2 групи по 2 особи. Одна група писатиме backend (серверну частину програми) і надаватиме документацію за запитами на сервер, друга група писатиме frontend (клієнтську частину). Потім групи зміняться місцями. Сподіваюся, що такий формат навчання буде цікавим та пізнавальним.

У першому модулі ми вивчатимемо HTML, CSS, Java Script і в кінці курсу ми підключимо до нашого проекту бібліотеку Bootstrap, адаптуємо для перегляду на різних пристроях, щоб наш проект почав виглядати дуже красиво). У процесі навчання ми навчимося працювати з системою керування версіями Git та опублікуємо свій проект на GitHub pages. Що це буде за проект? У процесі вивчення першого модуля ми напишемо та опублікуємо резюме та свій особистий довідник  за мовою розмітки гіпертексту HTML, мови опису зовнішнього вигляду CSS та основ однієї з найпопулярніших та затребуваних мов веб розробки — Java Script. Після закінчення курсу навчання буде проведено співбесіду із завданнями та питаннями щодо пройденого матеріалу для перевірки та закріплення отриманих знань.
