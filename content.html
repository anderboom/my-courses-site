Содержание курса

Порядок изучения материала
Модуль №1
Первое вводное занятие. Краткий обзор курса. Установка редактора кода. Основы работы с консолью браузера и командной строкой (1 занятие).
Изучаем основные HTML теги, делаем ДЗ (7 занятий)
Изучаем CSS. Использование Flexbox и Grids. Адаптивная вёрстка CSS Media Query (10 занятий). ДЗ по каждой теме занятия.
Создаём своё резюме, используя HTML и CSS (1 занятие)
Изучаем инструмент контроля версий GIT. Создаём аккаунт на платформе GitHub и публикуем его на GitHub Pages (1 занятие)
Изучаем основы языка программирования Java Script. ES6 синтаксис. Делаем много ДЗ по каждой теме занятий (8 занятий)
Учимся работать с DOM при помощи чистого Java Script, используя DOM API. Делаем ДЗ (6 занятий)
Практика Java Script. Методы массивов (2 занятия)
Практика Java Script. Функции в JS (2 занятия)
Изучаем формат JSON, как работать с данным форматом (1 занятие)
Изучаем работу с сетью. Что такое Промисы, Async/await синтаксис (2 занятия)
Подключаем к проекту библиотеку шаблонов Bootstrap, изучаем возможности данной библиотеки. Основная нагрузка на самомтоятельное изучение (1 занятие)
Закрепляем пройденный материал, пишем свой онлайн справочник и публикуем его на GitHub Pages (6 занятий)
Защищаем свой проект и проходим собеседование по пройденному материалу
